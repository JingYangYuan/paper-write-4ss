# 文献综述范文与提示词

本文档提供适用于各研究范式的通用文献综述结构模板和可复用写作提示词。两种范式的差异要点见文末 [范式差异速查](#四两种范式文献综述差异速查)。

---

## 一、通用四段式结构模板

### （一）核心概念界定与研究脉络 [L2-分节1]

{界定 1-2 个核心概念的定义、测量方式和学术渊源。简述该领域的研究发展脉络（按研究路径而非按时间），每处引文后必须插入作者自己的分析句 [L6]。}

关键句式：
- "{概念}是指{定义}。自{奠基性研究}以来，这一概念已成为{领域}的核心议题。"
- "早期研究主要关注{维度}……值得注意的是，这些研究虽然{贡献}，但主要停留在{层面}，较少{本文关注的维度}。"

### （二）既有研究的贡献与不足 [L2-分节2]

按研究路径（而非按国内外）组织对既有文献的批判性回顾，通常梳理 2-3 条主要路径：

**路径一：{路径名称}。** {贡献概述 + 2-3 篇代表性文献 + 具体不足}。

**路径二：{路径名称}。** {同上结构}。

**路径三：{路径名称}。** {同上结构}。

**总体评述**：上述路径分别揭示了{维度}，为本文提供了文献基础。然而，它们之间存在明显的分隔——{具体说明分隔在哪里}——尚未有整合性框架将{各维度}统一在同一理论脉络中。

### （三）中国情境的特殊性 [L2-分节3，可选]

{阐述中国情境的独特性以及既有中国研究对此议题的推进与不足。不是对西方研究的简单重复，而是指出中国情境的特殊制度/结构/文化因素如何使得该议题在中国呈现不同的面貌。}

### （四）分析框架建构与交汇 [L1][L4]

基于上述文献回顾，本文提出以下分析框架。首先，本文将{核心对象}分解为{2-3 个维度}。其次，本文在{既有理论}基础上引入{新因素/新维度}。再次，本文将这一分析置于{中国/特定情境}的脉络中。

根据这一分析框架，{一句话概括核心逻辑：X 通过 M 影响 Y，受 Z 的调节}。

## 二、文献综述提示词模板

### 2.1 理论框架型文献综述

```markdown
你是一位社会学/管理学核心期刊的资深作者。请根据以下要求撰写文献综述：

【研究主题】{填写}
【核心概念】{填写 1-2 个需要界定的核心概念}
【既有研究路径】{填写 2-3 条路径，每条附带 3-5 篇代表性文献}
【既有研究的主要不足】{填写 2-3 个具体不足}
【本文的分析框架】{填写框架的核心维度和逻辑}

写作要求：
1. 分为 2-4 个分节，以（一）（二）（三）（四）标注（[L2]）
2. 分节 1：概念界定 + 研究脉络概述
3. 分节 2：按研究路径组织批判性回顾（非按国内外组织）
4. 分节 3（可选）：中国情境特殊性
5. 分节 4：分析框架建构 + 各分节线索交汇到研究问题（[L1][L4]）
6. 每处引文后有思考痕迹，不得连续引用（[L6]）
7. 使用至少 3 种引文形式（[L5]）
8. 中英文文献混合，英文比例 ≥ 25%
9. 全文 3000-5000 字
```

### 2.2 假设推导型文献综述

```markdown
你是一位社会学/管理学核心期刊的资深作者。请根据以下要求撰写文献综述与研究假设：

【研究主题】{填写}
【核心变量关系】{填写自变量→因变量的关系}
【调节/中介变量】{填写}
【每条假设的核心逻辑与文献支撑】{填写}

写作要求：
1. 按逻辑线索（非时间或国别）组织文献回顾
2. 每回顾一条线索即推导出对应的研究假设（H1, H2, H3...）
3. 每条假设反向可追溯到文献综述中的具体引用
4. 文献回顾末尾有总体理论模型描述
5. 假设推导完成后过渡到方法部分
```

## 三、文献综述反模式检查清单

- [ ] **平行叙述不交汇**：各分节末尾是否都收束到研究问题？
- [ ] **文献装饰化**：是否区分了核心文献与外围文献？
- [ ] **"头重脚轻"**：文献综述引用密度是否远高于经验分析部分？（差异应 ≤ 2 倍）
- [ ] **以国内外分类**：是否按"国外研究-国内研究"机械罗列？
- [ ] **连续引用不分析**：是否有任何两个连续引用之间没有作者的分析句？
- [ ] **引文形式单一**：全文是否只用了 1-2 种引文呈现方式？（应 ≥ 3 种）
- [ ] **缺乏交汇段落**：文献综述最后是否出现了"基于上述分析"等收束信号词？

## 四、两种范式文献综述差异速查

| 维度 | 社会学研究范式 | 管理世界案例范式 |
|------|--------------|----------------|
| 组织方式 | 按研究路径/理论线索组织 | 按解释流派分类评述 |
| 结尾功能 | 建构分析框架 + 交汇到研究问题 | 指出不足 + 阐明本文理论切入 |
| 与中国情境的关系 | 作为特殊性补充（可选分节） | 作为核心论证要素（必须） |
| 典型结构 | 概念界定 → 路径回顾 → 不足分析 → 框架建构 | 流派一评述 → 流派二评述 → 流派三评述 → 总体不足 → 本文切入 |
| 常见字数 | 3000-5000 字 | 2000-4000 字 |
