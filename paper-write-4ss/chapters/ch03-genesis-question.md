# 第三讲：缘起与问题——追寻研究价值

## Core Idea
开头是论文的"领头羊"，最重要的任务是阐释研究价值（论证研究合法性）。好开头要精心设计第一笔、第一句和第一段。

## 开头的五要素
1. **引出现象**：学术现象或日常现象
2. **提出问题**：针对现象的矛盾性或结构力量提问
3. **定位知识脉络**：将现象放入学术领域
4. **论证合法性**：小型知识回顾 → 找缺口 → 正面阐述价值（核心任务）
5. **自我陈述**：告诉读者文章怎么写

两个层次的问题：前几个是描述性的（依靠经验材料可解决），最后一个是知识脉络级别的（引出文献综述和理论对话）。

## 两种开头风格

### 学术化开头
- 用凝练的学术话语概括现象，开门见山
- 确保开篇包含重要文献，设定学术基调
- 推荐初学者优先使用——不会沦为散文或记叙文
- 禁忌："随着5G技术的广泛运用""随着抖音在全国的普及"等大白话开头

### 场景化开头
- 从故事、新闻片段或生活场景进入
- 优点：可读性强、易引发共鸣
- 风险：学术感难建立；必须在场景后迅速转换到学术轨道
- 适用条件：场景特别契合 + 可以凝练概括 + 能快速转换

## 合法性论证的模式
1. **在视角上突破**：指出既有研究的视角局限（如"殖民者vs被殖民者"之外存在"殖民者之间的竞争"）
2. **"人无我有"思维**：证明某一议题在学术领域尚未得到充分关注——但必须在中观层次的学术领域中展现，不能仅在现象层面

## 自我陈述
包含研究对象、研究视角、研究材料和研究问题。可辅助系列性问题暗示后续主体结构。

## Key Takeaways
1. 开头是一篇论文中被阅读和修改次数最多的地方——"开头写顺了，后面往往一气呵成"
2. 转折词（然而、但是）是合法性论证的关键工具
3. 初学阶段优先使用学术化开头
4. "学问之道，在于学会提问"——提问是将现象定位到知识脉络的核心操作
5. 每一次合法性论证，都是在肯定和否定的辩证中对既有研究进行批判
