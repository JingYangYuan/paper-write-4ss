# 第十讲：投稿与写作伦理——走向公共的知识

## Core Idea
论文完成后，还要经过完善形式、投稿、修改、回应评审等环节才能变为公共知识。这些过程充满不确定性，需要定力和耐心。

## 一、完善论文形式

### 摘要
- 300-500字，类似小型论文
- 结构：研究合法性 → 研究对象/材料/问题 → 研究发现（详细、有层次） → 研究意义（凝练收尾）
- 实用技巧：与结尾的"总结全文"高度重合，略作修改即可

### 关键词
- 3-5个，兼顾专业性和广泛性
- 来源：研究背景（如"近代中国"）、研究对象（如"《人民日报》"）、学术概念（如"民族主义"）、涉及人群（如"老人传播"）

### 引用格式
- **脚注/尾注格式**（人文常用）：直观但占篇幅
- **APA格式**（社科常用）：正文作者+年份，文末参考文献列表
- 核心要求：准确性和前后一致性——"引注不是小事，而是学术素养和态度的试金石"

### 提交前最后两步
1. 打印纸质稿通读——纸质质感有助于发现电子版忽略的错误
2. 适当冷却——"陌生化"后再读，获得新审视

## 二、投稿策略

### 了解期刊风格
- 匹配文章与期刊风格可提高命中率
- 分散稿件去向，形成多元化发表状态
- 投稿后即投入下一个议题——已写完的文章是过去式

### 四种投稿结果
1. **快速退稿**：风格不适或水平不够，继续寻找更适合的期刊
2. **修改后再审**（最优）：刊登概率 > 50%，还能获得专业评审意见
3. **两位评审意见相左**：送第三位评审
4. **略作修改直接刊登**（罕见）：仍需认真对待所有意见

## 三、回应评审意见

### 评审意见的三分法
1. **格式细节问题**：仔细修改，容易完成
2. **阐释不清问题**：需要细致修改，但可以完成
3. **难以修改的实质问题**：真诚解释为什么没修改，或告知研究界限和后续打算

### 回复信的三原则
- **感谢评审付出**：绝大多数审稿是无偿劳动
- **逐条回复**：做了什么、怎么做的、为什么没做——这是学术对话和自我成长
- **不卑不亢、平等交流**：不需一味迎合，交流始终应是平等的

### 修改心态
- "白纸黑字"的敬畏——文章一旦发表就留下了永久记录
- 修改比写更痛苦，但这是完善文章的最后机会

## 四、写作伦理

### 引用伦理
- 少用转引——尽量找原文
- 多直接引用、少间接引用——凡有引用必有页码
- 引用准确——从前期摘录就做好记录

### 投稿伦理
- **一稿一投**：直到明确退稿或超过规定期限
- 中文期刊审稿周期约3个月，有的长达半年

### 署名伦理
- 人文社会科学遵循"谁主笔谁在前"
- 合作中的强势者不要侵占成果，弱势者要努力从客体变主体

## Key Takeaways
1. 摘要与结尾的"总结全文"高度重合——利用这个伏笔节省精力
2. 投稿前打印纸质稿通读，冷却后再次检查
3. 收到"修改后再审"是好消息——刊登概率已过半
4. 回复评审意见要逐条回应、分类处理、不卑不亢
5. 一稿一投是不可触碰的底线
