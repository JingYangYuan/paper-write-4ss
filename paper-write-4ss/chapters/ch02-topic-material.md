# 第二讲：写作的准备——想选题与找材料

## Core Idea
选题的好坏决定写作的价值和动力。好选题来自阅读与日常生活的"碰撞"，需要大小适中、具备创新持续性和社会关怀。

## 选题的三个来源
1. **阅读**：适用于理论化或历史类选题。在阅读中发现思想资源，如在对多位学者传播观念的比较阅读中发现"不可交流性 vs 追求共同性"两种对立传统。
2. **日常生活**：适用于经验研究。将日常体验学术化——如研究者自己就是金属乐爱好者、时间管理App重度用户。
3. **阅读与生活的碰撞**：最普遍的来源。阅读给予视角，生活带来经验，两者的一致或矛盾都可能催生选题。

## 好选题的三个标准

### 1. 大小适中的知识空间
- **向上**：可定位到有可观论述基础的知识脉络
- **向下**：可落实到有边界的、能搜集丰富材料的经验世界
- 警惕选题过于狭窄（如"拍一拍"——两头不靠）或选题过大（如"媒介时间的来临"——经验边界模糊）
- 关键思维：大事化小、小题大做，在大小之间伸缩

### 2. 创新与持续性
- **创新**：不要求开创全新领域，而是切入角度与现有研究不同；摆脱跟风心态，不满足于表面新现象
- **持续性**：有些选题只能做一篇文章，"不划算"；好的选题可以长期耕耘，形成个人学术标签

### 3. 社会关怀
- 精致的方法和完美的形式不能掩盖议题的空洞
- 深入社会之中，到具体人群中，发现真实困境
- 避免学术文章沦为文字游戏

## 自我追问六步法
**确定现象 → 定位知识脉络 → 阅读文献 → 寻找既有研究的缺口 → 论证研究合法性 → 查看经验材料能否支撑**

将抽象的选题判断转化为一系列可操作的步骤。每一步确认后，再重新评估文献和材料的匹配程度。

## 找材料
- 选题确定后，在题目范围内广泛搜集足量、新鲜、适合的材料
- 材料类型：阅读笔记/摘录、数据/问卷、史料/经验材料
- 抄写是一种有效整理方式——"有时候，慢即快"

## Key Takeaways
1. 选题不是灵感乍现，而是阅读和生活的持续碰撞
2. 内化自我追问思维，可独立判断选题可行性
3. 创新感是驱动写作的内在力量，不需要外在激励
4. 从新现象出发时，一定要想"新"背后的"旧"
5. 持续性选题 > 一次性选题，尤其在学位论文中
