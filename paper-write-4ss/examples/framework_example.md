# 分析框架范文与提示词

本文档提供两种研究范式中分析框架的结构模板和可复用的写作提示词。

---

## 一、分析框架在论文中的两种位置

| 位置 | 适用情况 | 写法要点 |
|------|---------|---------|
| **嵌入文献综述末尾** | 框架直接来自文献对话和综合 | 以"基于上述分析，本文提出以下分析框架"自然引出 |
| **独立设置"理论框架"节** | 框架有独立的理论来源或需较大篇幅展开 | 在文献综述后单独设节，系统阐述理论基础、维度和操作化 |

---

## 二、社会学研究范式分析框架

### 2.1 维度分解型（嵌入文献综述末尾）

**开头句**：基于上述文献回顾，本文提出以下分析框架。{研究对象/核心关系}可以从以下{2-3}个维度加以考察。

**维度一：{维度名称}。** 操作化为{具体指标}。根据{理论依据}，{该维度的核心假设}。

**维度二：{维度名称}。** 操作化为{具体指标}。已有研究发现{既有共识}，本文通过引入{新视角/新变量}，进一步分析{推进点}。

**维度三：{维度名称}。** 操作化为{具体指标}。这一维度弥补了既有研究过度侧重{已有维度}的不足。

**维度间关系**：{维度一}是{核心对象}最直接的{后果/表现}；{维度二}反映了中长期影响；{维度三}是前两者共同塑造的结果，同时产生反馈效应。

**调节/中介因素**：本文引入以下调节（或中介）变量：{变量一}（{简要说明}）、{变量二}（{简要说明}）。

**框架概括（一句话）**：{核心自变量}通过{中介变量}影响{因变量的三个维度}，受{调节变量}的调节。

---

### 2.2 理论对话型（独立成节）

将{2-3}种理论传统置于同一框架中对话：

**理论传统一：{理论名称}。** {核心洞见 + 对本文研究问题的适用性 + 该理论无法解释的方面}。

**理论传统二：{理论名称}。** {同上结构}。

**理论传统三：{理论名称}。** {同上结构}。

**理论交汇点**：将上述理论置于同一分析框架中，本文提出：{核心命题}。具体框架分层如下：
- 第一层（结构性因素）：{宏观制度/结构层面}
- 第二层（过程性因素）：{中观机制/互动层面}
- 第三层（后果性因素）：{微观结果/效应层面}

---

## 三、管理世界案例范式理论框架

### 3.1 机制链型（条件—行动—结果）

综合上述文献讨论，本文构建"条件—行动—结果"的整合性分析框架。

**条件维度。** {制度/技术/组织条件}构成了{研究对象}运作的基础场景：（1）……；（2）……；（3）……。

**行动维度。** 在给定条件下，行动者采取的策略包括：（1）……；（2）……。

**结果维度。** 条件与行动的交互产生了三重结果：（1）……；（2）……；（3）……（既有显性功能也有隐性功能）。

### 3.2 二维矩阵型

从 **X维度**（高/低）和 **Y维度**（强/弱）两个关键维度区分{研究对象}，形成四种理想类型：

|  | Y弱 | Y强 |
|------|------|------|
| **X高** | 类型A：{命名} | 类型B：{命名} |
| **X低** | 类型C：{命名} | 类型D：{命名} |

每种类型对应不同的运作逻辑。经验分析将围绕四种类型展开，重点考察类型间的转化条件。

## 四、分析框架提示词模板

### 4.1 社会学——维度分解型

```markdown
你是一位社会学研究者。请构建分析框架：

【研究问题】{填写}
【核心自变量】{填写}
【因变量的 2-3 个维度】{填写}
【调节/中介变量】{填写}
【依托的主要理论】{填写 1-2 个}

写作要求：
1. 以"基于上述分析，本文提出以下分析框架"开头
2. 将因变量分解为 2-3 个可操作的维度（每个维度有定义 + 指标 + 文献依据）
3. 阐述各维度间的逻辑关系（并列/递进/因果）
4. 说明调节/中介变量的作用机制
5. 框架的每个维度必须在后续经验分析中有对应小节
6. 用一句话概括框架核心逻辑
```

### 4.2 社会学——理论对话型

```markdown
你是一位社会学研究者。请构建跨理论对话分析框架：

【涉及的 2-3 种理论传统】{填写}
【每种理论的核心洞见与局限】{填写}
【理论交汇点】{填写}

写作要求：
1. 依次阐述每种理论的核心洞见及其适用性
2. 指出每种理论的局限
3. 在交汇点阐明各种理论如何互补构成完整框架
4. 可以分层（宏观→中观→微观）或分阶段组织
```

### 4.3 管理世界——机制链型

```markdown
你是一位《管理世界》风格的研究者。请构建"条件—行动—结果"机制链框架：

【研究现象/对象】{填写}
【条件因素】{填写 2-3 个}
【行动/策略】{填写}
【结果/产出】{填写 2-3 个层次}

写作要求：
1. 从既有文献中提炼各维度的理论依据
2. 阐述因果链条和反馈机制
3. 框架必须被案例数据支撑，不能先画模型再硬套材料
```

## 五、框架质量自检清单

- [ ] 框架的每个维度是否都有文献依据？
- [ ] 框架的每个维度是否在后续经验分析中有对应小节？
- [ ] 框架内部各维度关系是否清晰（并列/递进/因果/条件）？
- [ ] 框架复杂度是否与经验材料的信息量匹配？
- [ ] 框架核心逻辑是否可用一句话概括？
