# 研究发现（实证结果）范文与提示词

本文档提供两种研究范式的研究发现结构模板和可复用的写作提示词。

---

## 一、社会学研究范式：量化实证类

### 1.1 结构模板

**总起段 [E1]**：本节按照分析框架的{2-4}个维度——{维度一}、{维度二}、{维度三}——依次呈现{核心自变量}对{因变量}的影响。每个维度先报告基准结果，再引入{调节变量}进行分析。

---

**（一）{维度一标题}**

**[经验呈现]** 表 X 报告了{核心模型}的估计结果。{一句话概括表格内容}。主要发现：第一，{发现一 + 系数/效应量 + 显著性}。第二，{发现二 + 系数/效应量 + 显著性}。

**[分析阐释]** 上述结果具有两层含义。第一，{含义一——将统计结果翻译为实质性的社会含义}。第二，{含义二——与既有研究的发现进行对照}。

**[理论提炼]** 概括而言，{维度一}体现为"{核心概念命名}"——{一句话总结此维度的核心发现}。这一发现为{相关理论}提供了来自{中国/新情境}的经验证据。

---

**（二）{维度二标题} [E5-分节递进]**

在确认了{维度一的基本发现}之后，接下来的问题是：{引出下一维度的问题}。{维度二的标题}可以为此提供回答。

（随后按三层递进结构展开，同上）

---

**（三）{维度三标题}**

（同上结构）

---

### 1.2 量化实证提示词

```markdown
你是一位《社会学研究》的量化实证研究者。请根据以下要求撰写"经验分析"：

【分析框架的 2-4 个维度】{填写}
【每个维度的核心发现】{分别用 1-2 句话概括}
【主要表格/图】{列出编号和内容}
【关键调节/中介效应】{填写}

写作要求：
1. 分节标题可追溯到分析框架（[E1]）
2. 每个分节"经验呈现→分析阐释→理论提炼"三层递进（[E2]）
3. 正文先解释发现再呈现统计量（[E4]）
4. 不得只报告显著性不报告效应量（[E4]）
5. 分节间有递进标记："在此基础上""进一步而言"（[E5]）
6. 图表标题和注释有自足性（[E4]）
```

## 二、社会学研究范式：质性田野类

### 2.1 结构模板

**（一）{分析框架维度一的标题}**

**[引导段]** {描述经验场景或现象，引出本节的中心问题/主题}。根据田野观察和访谈，本文识别出{2-3 种典型模式/类型/机制}。

**[模式一的呈现]**

{引导句——交代经验材料的语境和来源}：

"{访谈引文——≤ 200 字}"（{受访者身份}，{访谈时间}）

{分析句——说明这段材料证明了什么，揭示什么机制，与既有研究的关系}。

**[模式二的呈现]**（同上结构）

**[模式三的呈现]**（同上结构）

**[本节理论提炼]** 综合以上分析，本文提炼出"{概念命名}"。它指涉的是：{一句话定义}。这一{概念/机制}不仅{对经验现象的解释}，也{对既有理论的对话/修正}。

---

### 2.2 质性田野提示词

```markdown
你是一位《社会学研究》的质性研究者。请根据以下要求撰写"研究发现"：

【分析框架的 2-4 个维度/阶段/机制】{填写}
【每个维度的核心发现】{分别用 1 句话概括}
【关键访谈/田野材料】{每个维度 2-3 段核心材料}

写作要求：
1. 每个分节"经验呈现→分析阐释→理论提炼"三层递进（[E2]）
2. 每条引文前有引导句（交代说话人身份和语境）（[E3]）
3. 每条引文后有分析句（说明材料证明了什么）（[E3]）
4. 每条引文 ≤ 200 字（[E3]）
5. 分节间有递进关系标记（[E5]）
6. 每个分节末尾提炼核心概念/机制
```

## 三、管理世界案例范式：研究发现

### 3.1 三种组织方式

**方式一：阶段-机制型**

```
（一）第一阶段：{阶段命名}——{核心机制命名}
  证据链呈现（表格/引文/档案）→ 机制阐释 → 小结

（二）第二阶段：{阶段命名}——{核心机制命名}
  （同上）

（三）第三阶段：{阶段命名}——{核心机制命名}
  （同上）
```

**方式二：维度-类型型**

```
（一）{维度一}：{类型命名}
  证据呈现 → 分析 → 类型特征提炼
（二）{维度二}：{类型命名}
（三）{维度三}：{类型命名}
（四）类型间关系与转化
```

**方式三：机制-张力型**

```
（一）{正向机制}：{机制命名与阐释}
（二）{内在张力/矛盾}：{张力表现与根源}
（三）{边界条件/情境因素}：{机制生效的条件}
```

### 3.2 管理世界案例提示词

```markdown
你是一位《管理世界》风格的案例研究者。请撰写"案例分析/研究发现"：

【案例名称】{填写}
【组织方式】{阶段-机制型 / 维度-类型型 / 机制-张力型}
【核心机制/类型】{填写 2-4 个，每个需有具体命名}
【证据类型】{填写}

写作要求：
1. 每个部分围绕一个核心机制展开，机制需有具体命名
2. 用表格呈现关键证据链（信息源+核心内容+支撑的命题）
3. 阐释：表现形态→运作逻辑→产生结果
4. 引文和表格交叉使用
5. 不写事件流水账，按机制逻辑组织
6. 每个机制末尾概括其理论含义
```

## 四、研究发现质量自检清单

### 通用检查

- [ ] 分节标题可否追溯到分析框架？
- [ ] 各分节间是否有递进关系标记？
- [ ] 每个分节开头有引导句、结尾有小结句吗？
- [ ] 经验材料比例 ≤ 50%？

### 量化实证额外检查

- [ ] 图表标题和注释是否自足？
- [ ] 正文是否先解释发现再呈现统计量？
- [ ] 是否报告了效应量而非仅报告显著性？

### 质性田野额外检查

- [ ] 每条引文前后是否有引导句和分析句包裹？
- [ ] 单条引文是否 ≤ 200 字？
- [ ] 引文选择是否有代表性？

### 管理世界案例额外检查

- [ ] 是否避免了事件流水账？
- [ ] 每个机制是否有明确命名？
- [ ] 是否有证据链表格？
- [ ] 案例分析是否为理论贡献做好了铺垫？
