# 引言（问题的提出）范文与提示词

本文档提供两种研究范式的引言结构模板和可复用的写作提示词。

---

## 一、社会学研究范式引言

### 1.1 结构模板（五段式，带标准标注）

**第一段：现象/背景引入 [I1]**

{学术现象、政策数据或理论争议的具体陈述。量化研究：以政策文件、统计数字或宏观趋势开篇。质化研究：以学术现象、理论争议或经验困惑开篇。5 句内出现可验证的陈述。}

**第二段：转折 + 既有研究不足 [I2]**

然而/但是/不过，既有研究对{具体议题}仍缺乏充分的{经验刻画/理论解释/机制分析}。现有研究多集中于{已有方向}，少有的几项{相关研究}也主要关注{已有维度}，对{本文关注的缺口}关注不足。

**第三段：缺口具体化 + 切入点 [I3][I4]**

不仅如此，既有研究多聚焦于{宏观层面}，较少关注{本文聚焦的具体维度/机制/对象}。特别是当{具体情境/条件}时，{核心问题}是否以及在多大程度上影响了{结果}，既有研究尚未给出充分的{经验/理论}回答。{切入点}正是观察这一问题的恰当棱镜，因为{选择理由——至少 2 句}。

**第四段：研究贡献陈述 [I5]**

基于上述分析，本文尝试在以下三个方面做出贡献。首先，在经验层面，{具体经验贡献}。其次，在{分析策略/概念/方法}层面，{具体推进}。最后，在理论层面，{理论贡献}。

**第五段：研究自我陈述 [I6][I7]**

本文使用{数据来源/田野材料}，运用{研究方法}，系统分析{研究内容}。{注意：交代研究对象、方法和数据，不得以"本文结构如下"代替。}

---

### 1.2 社会学范式引言提示词

```markdown
你是一位《社会学研究》的资深作者。请根据以下要求撰写引言：

【研究主题与核心问题】{填写，1-2句话}
【经验材料】{数据来源/田野地点/研究方法}
【既有研究缺口】{填写 2-3 个具体到维度/视角/对象层面的不足}

写作要求：
1. 开篇用具体的学术现象、政策数据或理论争议引入（[I1]）
2. 第 1-3 段内使用"然而/但是/不过"转折，引出既有研究不足（[I2]）
3. 研究缺口必须具体，不得只说"现有研究较少"（[I3]）
4. 明确陈述切入点和选择理由（[I4]）
5. 用"首先...其次...最后..."阐述至少 3 点研究贡献（[I5]）
6. 结尾交代研究对象、数据和方法，不以"结构如下"代替（[I6][I7]）
7. 全文 1500-2500 字
```

## 二、管理世界案例研究范式引言

### 2.1 结构模板（五段式）

**第一段：国家战略或重大现实背景**

近年来，随着{国家战略/重大政策}的深入推进，{研究对象/现象}已成为理论界与实践界共同关注的重要议题。{具体政策文件引用或统计数字}表明……这一趋势深刻影响着{相关领域/行业/社会}。

**第二段：现实中的矛盾、悖论或异常现象**

然而，在实践中出现了一个值得深入解释的现象：一方面，{现象A/预期效果}；另一方面，{现象B/意外结果}。这一看似矛盾的现象表明，{研究对象}的运行逻辑并非如既有认知那样简单，其背后可能存在尚未被充分揭示的机制。

**第三段：既有研究的解释路径与不足**

已有研究主要从以下视角解释上述现象。第一类研究强调{视角一}，揭示了{贡献}，但难以解释{特定现象}。第二类研究关注{视角二}，提供了{贡献}，但对{特定维度}关注不足。第三类研究从{视角三}出发，有助于理解{贡献}，但仍未能充分揭示{核心机制}。总体来看，既有研究至少存在两个方面的不足：其一，{理论层面的缺口}；其二，{经验层面的空白}。

**第四段：研究问题**

由此，本文试图回答如下核心研究问题：{研究问题}。这一问题的回答不仅有助于揭示{研究对象}的内在机制，也有助于推进{相关理论}在中国情境下的发展。

**第五段：案例选择、方法与理论贡献**

为回答上述研究问题，本文选取{案例名称}作为{单案例/多案例/纵向案例}研究对象（选择理由：{典型性/极端性/启发性}）。在方法上，本文通过{数据来源}等方式收集研究资料。

本文可能的理论贡献在于：第一，{新概念/新机制}；第二，{修正既有假设/拓展边界}；第三，{中国情境特殊机制/过程模型}。

---

### 2.2 管理世界范式引言提示词

```markdown
你是一位熟悉《管理世界》期刊风格的案例研究写作顾问。请根据以下要求撰写引言：

【研究对象/现象】{填写}
【国家战略或重大政策背景】{填写}
【现实中的矛盾或悖论】{填写}
【既有研究的 2-3 种解释路径】{填写}
【既有研究的核心不足】{填写 2 个具体缺口}
【核心研究问题】{填写}
【案例名称与类型】{填写}

写作要求：
1. 第一段以国家战略或重大现实问题开篇
2. 第二段呈现现实中的矛盾或悖论，制造理论张力
3. 第三段分类评述既有研究的贡献与不足
4. 第四段明确提出研究问题
5. 第五段交代案例选择、方法和理论贡献
6. 全文 2000-3000 字
```

## 三、两种范式引言差异速查

| 维度 | 社会学研究范式 | 管理世界案例范式 |
|------|--------------|----------------|
| 开篇方式 | 学术现象/政策数据/理论争议 | 国家战略/重大现实问题 |
| 张力来源 | 既有学术文献的缺口和争议 | 现实中矛盾、悖论或异常现象 |
| 贡献陈述重点 | 对学术知识的推进 | 对理论与实务的双重贡献 |
| 结尾方式 | 研究对象+方法+数据 | 案例选择+方法+理论贡献 |
| 常见字数 | 1500-2500 字 | 2000-3000 字 |

---

## 四、引言写作通用禁忌（两种范式均适用）

1. 不得以"随着……的发展"开头但没有后续具体信息
2. 不得只说"现有研究较少"而不指出少在哪个维度、哪个层面
3. 不得以"本文结构如下：第二部分……第三部分……"作为引言结尾
4. 不得在引言中宣称"本文首次……""本文填补了……空白"
5. 不得连续引用多篇文献而不插入作者自己的评价和分析
