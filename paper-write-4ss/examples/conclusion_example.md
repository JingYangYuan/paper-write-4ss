# 结论与讨论范文与提示词

本文档提供两种研究范式的结论与讨论结构模板和可复用的写作提示词。

---

## 一、社会学研究范式结论

### 1.1 六段式结构模板（带标准标注）

**第一段：总结全文 [C1]**

本文基于{数据/田野材料}，运用{研究方法}，系统考察了{研究内容}。研究发现可以概括为以下三点。

第一，{发现一 + 核心机制/效应命名 + 关键数据/证据}。
第二，{发现二 + 核心机制/效应命名 + 关键数据/证据}。
第三，{发现三 + 核心机制/效应命名 + 关键数据/证据}。

总体而言，本文的研究发现表明，在{宏观背景}下，{核心变量/机制}已成为影响{结果}的关键因素。它不仅塑造了{当下影响}，还通过{渠道}影响了{长期后果}。

**第二段：纵横延展 [C2]**

将本文的发现放入更广阔的时空框架中审视。

在时间维度上，本文所揭示的"{核心发现}"现象，与{既有文献脉络}既有呼应也有差异。呼应的方面在于：{相同之处}。差异的方面在于：在中国情境中，{特殊性}。

在空间维度上，将本文发现与其他{地区/领域/国家}进行比较。{比较对象}呈现{特征}，而中国情境呈现{不同特征}。这一比较暗示，{发现}不仅取决于{因素一}，还取决于{因素二——制度/结构/文化}的调节作用。

**第三段：研究视角的独特性 [C3]**

本文从{具体视角}切入{研究议题}，这一视角的独特价值在于：{具体说明——不是泛泛说"提供了新视角"，而是说清楚这个视角揭示了哪些其他视角看不到的东西}。

**第四段：解决遗留问题 [C4]**

正文中有{1-2}个问题尚未充分展开。第一个遗留问题是{具体问题 + 为何重要 + 可能的解释方向}。第二个遗留问题是{同上}。

**第五段：研究限制 [C5]**

本文存在以下研究限制。第一，{具体限制一——测量/样本/方法层面的局限}。第二，{具体限制二}。第三，{具体限制三}。

**第六段：以未来设想结尾 [C6]**

后续研究可以从以下几个方面推进。首先，在方法上，{改进方向}。其次，在研究范围上，{拓展方向}。最后，在比较视角上，{跨领域/跨国家比较方向}。这些方向的研究不仅有助于深化对{议题}的理解，也有助于{更广泛的理论/实践意义}。

### 1.2 社会学范式结论提示词

```markdown
你是一位《社会学研究》的资深作者。请撰写"结论与讨论"：

【研究对象与视角】{填写}
【方法与数据】{填写}
【核心研究发现】{至少 3 点，有层次地描述}
【时空延展方向】{时间维度 + 空间维度各 1 个比较对象}
【遗留问题】{1-2 个正文不便展开的问题}
【研究限制】{2-3 条具体限制}
【未来设想】{1-3 个后续研究方向}

写作要求：
1. 总结：对象+方法+发现有层次地描述（≥ 3 点）+ 意义（[C1]）
2. 纵横延展：时间+空间维度的比较讨论（[C2]）
3. 视角独特性（[C3]）
4. 遗留问题（[C4]）
5. 研究限制（≥ 2 条具体限制）（[C5]）
6. 以未来设想结尾——全文最后一段不以限制收尾（[C6]）
7. 体量 ≥ 全文的 5%
```

## 二、管理世界案例范式结论

### 2.1 四段式结构模板

**第一段：研究总结（理论发现而非经验归结）**

本文以{案例名称}为研究对象，通过{案例类型}研究，揭示了{研究对象}的{核心机制/阶段/类型}。研究发现可以概括为以下三点。第一，{机制/概念一}：{核心论述}。第二，{机制/概念二}：{核心论述}。第三，{机制/概念三}：{核心论述}。这些发现共同指向{整合性的理论命题}。

**第二段：理论贡献（再度确认和深化）**

本文的理论贡献主要体现在三个方面。首先，在{理论脉络一}方面，本文{具体推进/修正/拓展}。其次，在{理论脉络二}方面，本文{具体推进}。最后，在更广泛意义上，本文{对中国情境的理论启示}。

**第三段：实践启示与政策含义**

本文的研究发现对于{实践领域/政策制定}具有以下启示。第一，{具体建议}，而非{常见做法}。第二，需要关注{具体方面}，建立{机制/制度}以{功能}。第三，{建议}。（实践启示必须基于前文的理论分析，不能脱离研究发现空谈。）

**第四段：研究局限与未来展望**

本文也存在一定的局限性。第一，{局限一}。第二，{局限二}。未来的研究可以{方向一}、{方向二}等，以进一步深化对{研究主题}的认识。

---

### 2.2 管理世界范式结论提示词

```markdown
你是一位《管理世界》风格的案例研究者。请撰写"结论与启示"：

【案例名称与类型】{填写}
【核心机制/类型/模型】{2-4 个，每个需有命名和定义}
【理论贡献】{2-3 条，"既有……但……本文……因此……"格式}
【实践启示】{2-3 条具体建议}
【研究局限】{2 条具体局限}

写作要求：
1. 以理论发现总领全篇
2. 理论贡献与正文呼应但不重复（此处更宏观）
3. 实践启示基于研究发现，不空谈
4. 以积极展望收尾
```

## 三、两种范式结论差异速查

| 维度 | 社会学研究范式 | 管理世界案例范式 |
|------|--------------|----------------|
| 总结重点 | 有层次呈现研究发现（≥ 3 点） | 提炼核心机制/模型的理论含义 |
| 延展方向 | 学术延展：与其他领域/国家/时期比较 | 实践延展：政策含义、管理启示 |
| 特色段落 | 研究视角独特性 [C3]、遗留问题 [C4] | 实践启示与政策含义 |
| 常见字数 | 正文的 5%-10% | 正文的 5%-8% |

---

## 四、结论写作反模式（两种范式均适用）

- **结尾体量过小**：3 万字论文只有半页结论 → 结论 ≥ 全文 5%
- **以研究限制结尾**：最后一段以"本文存在以下不足"收场 → 必须以未来展望结尾
- **空洞限制表述**："本文存在诸多不足" → 必须说出 2-3 条具体限制
- **总结过于空泛**："具有重要意义" → 必须具体说出发现了什么
- **结论引入新材料**：忽然引用前文未出现的文献 → 结论应围绕已有发现展开
