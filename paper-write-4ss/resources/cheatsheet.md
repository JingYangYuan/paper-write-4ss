# 快速参考 v2.0

## 论文七要素 + 质量门槛

| 要素 | 核心任务 | 通过标准（硬性门槛） | 常见反模式 |
|------|---------|---------------------|-----------|
| **标题** | 凝练全文精华 | 主标题 ≤ 20 字 + 副标题 ≤ 25 字；主标题含发现/概念/问题三类信息之一 | "以...为例"套路化；标题与正文发现不一致 |
| **摘要** | 五要素浓缩 | 背景→方法→发现（具体有层次）→意义→展望；发现部分 ≥ 3 个具体点 | "本文对...进行了分析"空泛表述 |
| **引言** | 论证研究合法性 | [I1]-[I7] 七条全部通过 | 只说"现有研究较少"不具体指出缺口；以"结构如下"结尾 |
| **文献综述** | 建立分析框架 | [L1]-[L6] 六条全部通过；终末有交汇段落 | 平行叙述不交汇；文献装饰化；引文后无分析 |
| **方法** | 叙事性呈现研究过程 | 质性：田野时间/地点/进入路径/对象表格；定量：数据/样本/变量/理由 | 笼统说"进行了深度访谈"不交代具体信息 |
| **经验分析** | 逻辑组织材料 | [E1]-[E5] 五条全部通过；分节递进有标记；每条材料有分析跟进 | 连续堆砌材料；分节纯并列无递进 |
| **理论对话** | 回到知识脉络提炼贡献 | [T1]-[T5] 五条全部通过；有比较性讨论 | 另起炉灶不回到引言脉络；空泛宣称"填补空白" |
| **结论** | 总结+延展+展望 | [C1]-[C6] 六条全部通过；全文最后一段不以限制结尾 | 结尾体量过小（< 全文 5%）；以弱点收尾 |

## 经验材料组织逻辑决策表

| 材料特点 | 推荐逻辑 | 典型操作 | 递进标记词 | 参考论文 |
|---------|---------|---------|-----------|---------|
| 有明确时间线索 | 内在时间节点 | 深入材料找内在分期，不用政治时间 | 早期→中期→近期 | 董浩（2026）1980年代以来的婚姻变迁 |
| 有空间/地理差异 | 空间分层 | 按地理层级递进：城市→县域→乡村 | 在这一层级上...进一步延伸到... | 沙垚（2026）县域→乡村→社区 |
| 涉及过程/事件 | 事件化/过程化 | 分解为起因→经过→结果 或 前→中→后 | 起初...随后...最终... | 宋锴业（2026）数字社会挑战→算法应对→三重逻辑 |
| 有开放型理论参照 | 理论层次 | 局部借用理论框架，按框架自身层次组织 | 在XX层面...在更深层的YY层面... | 张文宏（2026）概念→机制→范式 |
| 以上都不适用 | 人为建构 | 详细交代划分依据和认知合理性 | 出于以下考虑，本文按...划分... | 马文（2026）四种进路递进 |

## 文献综述写作决策表

| 情况 | 适用方式 | 关键操作 | 通过标准 | 参考论文 |
|------|---------|---------|---------|---------|
| 涉及 2-3 个独立领域 | 横向铺陈 + 终末交汇 | 分述各方→暗线指向→末尾交汇 | 交汇段落引用前文各分节的关键发现 | 董浩（2026）年龄匹配+教育匹配→交汇 |
| 只有一个核心领域 | 纵向延伸/"下台阶" | 宏观轮廓→中观议题→微观临近研究 | 每层都引用具体文献，问题逐层聚焦 | 马文（2026）描述的学脉→式微→重启→深描 |

## 全文质量检查五轮序列

### 第一轮：结构完整性（5 分钟）
- [ ] 七大要素齐全
- [ ] 各部分篇幅比例合理（经验分析 30-40%）
- [ ] 目录逻辑连贯（只看标题可判断骨架）

### 第二轮：论证严密性（15 分钟）
- [ ] 引言：转折词 + 缺口 + 贡献 + 自我陈述
- [ ] 文献综述：框架 + 交汇 + 引文多样性
- [ ] 经验分析：分节递进 + 三层分析
- [ ] 理论对话：回到引言脉络 + 比较讨论
- [ ] 结论：六要素覆盖 + 不以限制结尾

### 第三轮：材料与引文（10 分钟）
- [ ] 引文形式 ≥ 3 种
- [ ] 每处引文后 ≥ 1 句分析
- [ ] 文献引用前后密度均衡（差异 ≤ 2 倍）
- [ ] 中英文混合引用

### 第四轮：语言规范（10 分钟）
- [ ] 无"一文中""该文"
- [ ] 标题无"以...为例"
- [ ] 无空泛创新宣称
- [ ] 无未定义缩写
- [ ] 无正文中嵌入完整论文标题

### 第五轮：格式体例（10 分钟）
- [ ] 参考文献格式一致
- [ ] 图表标题自足
- [ ] 正文引用与文献列表一一对应
- [ ] 打印通读 + 冷却 48h

## 论文检查清单（提交前）

- [ ] 标题：正副结构，无套路化表达，与正文发现一致
- [ ] 摘要：五要素齐全，发现具体（≥ 3 点）
- [ ] 引言：七步链条完整，有转折词，有 3 点以上贡献陈述，以"本文..."收尾
- [ ] 文献综述：有分析框架，分节有交汇，至少 3 种引文形式
- [ ] 方法：叙事清晰，选择理由充分，对象/样本信息完整
- [ ] 经验分析：分节递进有标记，材料有分析跟进，每节有小结
- [ ] 理论对话：回到引言脉络，有比较性讨论
- [ ] 结论：六要素覆盖，末段不以限制结尾，体量≥ 全文 5%
- [ ] 参考文献：完整一致，正文与列表对应，中英混合
- [ ] 冷却检查：放置 48h 后通读 + 纸质稿标注

## 投稿四结果应对表

| 评审结果 | 概率 | 含义 | 应对策略 |
|---------|------|------|---------|
| 快速退稿 | 约 40% | 风格不适或水平不够 | 保持平常心，分析原因，投更合适的期刊 |
| 修改后再审 | 约 30% | 刊登概率 > 50% | 逐条回应，认真修改，修改说明 ≥ 修改本身的工作量 |
| 两位意见相左 | 约 20% | 分歧大，需第三方 | 等待第三位评审意见，先着手处理共识部分 |
| 略作修改直接刊登 | 约 10% | 罕见 | 认真对待每一条意见，不因"通过"而轻慢 |
