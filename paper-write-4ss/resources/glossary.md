# 术语表 v2.0

## 一、论文要素术语

- **七大要素** — 一篇完整学术论文的构成部分：标题+摘要+关键词 → 引言（问题的提出）→ 文献综述/理论框架 → 研究方法与数据 → 经验分析/研究发现 → 理论对话 → 结论与讨论。缺一不可。

- **合法性论证** — 引言部分的核心任务，通过七步链条（现象→问题→脉络→回顾→缺口→价值→自我陈述）证明研究值得做、必须做。转折词（然而/但是/不过）是关键工具。

- **自我陈述** — 引言末尾交代"本文用什么材料/方法研究什么问题"的段落。区别于论文结构预告（"本文分为五部分..."），后者应避免。

- **分析框架** — 在文献综述中建构的、用于指导经验分析的概念架构。分析框架的分维度应与经验分析的分节标题一一对应。

- **交汇段落** — 横向铺陈式文献综述末尾的收束段落，将所有分节的讨论线索统一到本文的研究问题上。

- **知识脉络** — 围绕某议题的系列学术论述的总体。理论对话必须回到引言中最初定位的知识脉络。

## 二、写作技法术语

- **三层递进** — 经验分析各分节的组织结构：第一层呈现材料 → 第二层分析阐释 → 第三层理论提炼。

- **引-析-结三段式** — 质性材料呈现的标准模板：引前句（身份+语境）→ 引文（≤ 200 字）→ 析后句（揭示意涵）。

- **下台阶** — 文献综述的纵向递进策略：从宏观到中观到微观，每一步都增加研究问题的聚焦度和合法性。

- **比较性延展** — 理论对话中将个案发现与至少 2 个外部领域/研究进行比较，从特殊上升到普遍。

- **纵横交错** — 文献综述同时兼具横向广度（多个领域）和纵向深度（宏观→微观）的最佳实践方式。

- **冷却检查法** — 投稿前将论文放置 48 小时以上，以"陌生读者"眼光重新通读，以发现盲点。

## 三、反模式术语（出现即需修正）

- **文献装饰化** — 反模式：文献综述只为装点门面，不分轻重地罗列文献，不指向自己的研究。

- **平行不交汇** — 反模式：横向铺陈的各分节结尾没有收束到研究问题，读者不知道脉络之间的关系。

- **头重脚轻** — 反模式：文献综述引用密度远高于后续经验分析部分，论文前后学术感失衡。量化标准：前后引用密度差异 > 2 倍。

- **空泛宣称** — 反模式：说"既有研究较少关注"而不具体指出在哪个维度上少、少在何处。

- **虚假创新宣称** — 反模式：使用"本文首次..."、"本文开创性地..."等表述，除非确为全新范式。

- **目录式结尾** — 反模式：引言最后以"本文第二部分介绍研究方法，第三部分...第四部分..."代替实质性的自我陈述。

## 四、系统操作术语

- **硬性标准（Hard Gate）** — 论文各部分必须满足的最低通过条件，以 [编号] 标记（如 [G1]、[I1]）。不通过则该部分需重写。所有硬性标准均来自对《社会学研究》等期刊发表论文的归纳。

- **IRON RULE（铁则）** — 绝对不可违反的写作禁令。违反即判该部分不合格。在 patterns.md 中以 `IRON RULE` 明确标记。

- **通过条件** — 每个硬性标准附带的可操作验证方式。通常为文本搜索指令或计数检查。

- **迭代规则** — 每个写作阶段的多轮修改建议。一稿（框架）→ 二稿（填充）→ 三稿（打磨），每轮有明确的检查重点。

- **质量门槛（Quality Gate）** — 论文提交前的五轮逐项检查序列：结构完整性 → 论证严密性 → 材料与引文 → 语言规范 → 格式体例。

## 五、研究方法术语

- **进入路径** — 质性研究中研究者如何进入田野现场的具体渠道和方式，方法部分必须交代。

- **观察姿态** — 研究者在田野中的角色：参与式观察/非参与式观察/潜水/混合。

- **三角互证** — 用多种来源的材料（访谈+观察+二手资料+文献）相互验证研究发现。

- **操作化** — 将抽象概念转化为可测量的具体指标。

- **rank-rank 相关系数** — 基于相对地位百分位的非参数相关系数，用于跨时期可比的婚姻匹配研究（董浩，2026）。

- **交换指数（Exchange Index）** — 基于反事实框架的匹配估计量，用于量化多维地位交换（董浩，2026；Xie & Dong, 2021）。

- **数据深描** — 运用多模态数据与算法技术对社会现象的时空结构与潜在关系进行清晰呈现的方法（马文、陈云松，2026）。

## 六、投稿术语

- **一稿一投** — 同一时间论文只能投给一份期刊的学术伦理原则。

- **修改后再审** — 最优评审结果，刊登概率 > 50%。需逐条回复评审意见并标注修改位置。

- **评审意见三分法** — 将评审意见分为：格式细节（快速修改）、阐释不清（细致修改）、实质问题（真诚解释）三类，分别制定应对策略。

- **回复信三原则** — 感谢评审付出 → 逐条回复（做了什么/怎么做的/为什么没做）→ 不卑不亢平等交流。

## 七、本 Skill 来源文献

本术语表中新增的系统和操作类术语（硬性标准、IRON RULE、通过条件、迭代规则、质量门槛等）为本 Skill v2.0 的系统设计概念，用于将学术写作标准操作化。论文要素类、写作技法类、反模式类术语来自对《学术写作十讲》（卞冬磊）的提炼和对《社会学研究》发表论文的结构分析。
