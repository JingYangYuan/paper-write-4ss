# 两种研究范式的对比与选择指南

当用户开始论文写作时，首先需要明确论文属于哪种研究范式。本指南区分两种在中国社会科学领域最具代表性的写作范式，帮助用户选择合适的写作路径。

---

## 一、两种范式的核心差异

| 维度 | 社会学研究范式 | 管理世界案例研究范式 |
|------|--------------|-------------------|
| **目标期刊** | 《社会学研究》《社会》《中国社会科学》等 | 《管理世界》《公共管理学报》《中国行政管理》等 |
| **驱动逻辑** | 学术脉络中的知识缺口 → 经验困惑 → 实证分析 → 理论对话 | 中国重大现实问题 → 理论反常/现实悖论 → 案例研究 → 机制建构 |
| **研究问题来源** | 既有学术文献中的未解问题、概念张力、方法局限 | 国家战略背景下的实践困惑、制度悖论、组织反常现象 |
| **理论角色** | 对话者：在知识脉络中定位贡献，回到文献脉络 | 建构者：从案例中提炼新概念、机制、模型或类型学 |
| **方法偏好** | 量化实证（调查数据、计量模型）、质性田野（民族志、深度访谈）、混合方法、理论分析 | 单案例/多案例/纵向案例/嵌入式案例研究，强调一手访谈与档案资料 |
| **经验材料呈现** | 数据结果表格 + 统计推断；访谈片段 + 三层分析递进 | 阶段-机制链条 + 概念命名 + 证据链展示（表格+引文+编码） |
| **理论贡献表述** | "本文尝试..."、"对...提供了新的经验证据"、"有助于深化对...的理解" | 提出新概念/修正既有假设/构建过程模型/形成类型学/揭示中国情境特殊机制 |
| **引言开篇** | 学术现象或政策数据开篇，迅速定位知识脉络 | 国家战略或重大现实背景开篇，强调实践重要性 |
| **结论风格** | 总结发现 + 纵横延展 + 研究限制 + 未来展望 | 总结机制模型 + 理论贡献再确认 + 实践启示 + 政策含义 |

## 二、如何选择：决策流程

### 第一步：判断研究问题的来源

```
你的研究问题主要从哪里来？
├── 从学术文献的缺口、争议或概念张力中来
│   → 社会学研究范式
│
├── 从中国重大现实问题、政策实践或组织反常现象中来
│   → 管理世界案例研究范式
│
└── 两者兼有，难以判断
    → 继续第二步
```

### 第二步：判断经验材料的性质

```
你的经验材料是什么类型？
├── 大规模调查数据（CFPS/CGSS/CSS 等）、统计数据、实验数据
│   → 社会学研究范式（定量路径）
│
├── 田野调查、深度访谈、参与观察、历史档案
│   ├── 目标是描述机制/过程 + 与学术脉络对话
│   │   → 社会学研究范式（质性路径）
│   └── 目标是提炼管理/组织/治理机制或模型
│       → 管理世界案例研究范式
│
├── 单一或多个组织的案例深度研究
│   → 管理世界案例研究范式
│
└── 理论/概念分析，不依赖一手经验材料
    → 社会学研究范式（理论分析路径）
```

### 第三步：判断目标期刊

```
你打算投哪个期刊？
├── 《社会学研究》《社会》《中国社会科学》
│   → 社会学研究范式
│
├── 《管理世界》《公共管理学报》《中国行政管理》《公共行政评论》
│   → 管理世界案例研究范式
│
├── 《社会学评论》《青年研究》《妇女研究论丛》等专业期刊
│   → 社会学研究范式（可根据具体期刊微调）
│
└── 尚未确定目标期刊
    → 根据前两步的判断选择范式
```

## 三、两种范式的论文结构差异

### 社会学研究范式（七要素标准结构）

```
标题 + 摘要 + 关键词
引言（问题的提出）—— 论证研究合法性，定位知识脉络
文献综述 / 理论框架 —— 建立分析框架，为研究铺路
研究方法与数据 —— 叙事性呈现研究过程
经验分析 / 研究发现 —— 以逻辑组织材料，三层递进
理论对话 —— 回到知识脉络，提炼理论贡献
结论与讨论 —— 总结全文，延展意义
```

### 管理世界案例研究范式

```
标题 + 摘要 + 关键词
引言 —— 国家战略背景 → 现实悖论 → 研究缺口 → 研究问题 → 案例与贡献
文献综述与理论框架 —— 分类评述 → 不足 → 本文切入 → 分析框架建构
研究设计 —— 方法选择理由 + 案例选择理由 + 数据收集 + 编码过程 + 信度效度
案例分析/研究发现 —— 阶段-机制 / 维度-类型 / 机制-张力
理论贡献 —— 对话式贡献，每条按"既有-不足-本文-推进"格式
结论与启示 —— 理论贡献再确认 + 实践启示 + 政策含义 + 不足与展望
```

## 五、两种范式的常见误区

### 社会学研究范式的误区

1. **空谈"社会转型"**：用宏观背景代替具体的学术脉络定位
2. **理论与经验两张皮**：分析框架在文献综述中搭建好但在经验分析中不出现
3. **文献综述沦为书目清单**：没有交汇段落，没有为研究铺路
4. **引言以"结构如下"结尾**：用论文结构预告代替研究内容陈述
5. **经验分析只有"呈现"没有"分析"**：堆砌数据或引文但不解说

### 管理世界案例范式的误区

1. **写成经验总结或工作汇报**：缺乏理论抽象，只是对案例的叙述
2. **政策建议替代理论贡献**：结尾只有"应该怎么做"没有"我们知道了什么"
3. **概念空转**：用复杂术语包装简单现象，没有实质理论推进
4. **理论框架先于案例**：先画模型再用案例硬套，削足适履
5. **案例选择理由不充分**：没有论证案例的典型性、极端性或理论适配性

## 六、范式转换：何时及如何

有时候，用户提供的论文可能更接近另一种范式。以下情况建议转换：

| 当前状态 | 建议转换方向 | 理由 |
|---------|------------|------|
| 用管理世界范式写纯学术理论对话 | → 社会学研究范式 | 问题来源是学术脉络而非实践 |
| 用社会学研究范式写单一组织案例 | → 管理世界范式 | 材料更适合机制建构型案例研究 |
| 两种范式的结构混杂 | → 选定一种并统一 | 评审会按一种范式标准评价 |

转换时的关键操作：
- 引言重写：改变开篇方式和合法性论证逻辑
- 文献综述重构：改变组织方式和交汇方式
- 结论调整：改变延展方向（学术延展 vs 实践启示）
